Beschreibung

Im Webdesign spielen mehrere Komponenten zusammen um eine Webseite zu erstellen. Diese sind HTML, CSS, JavaScript und PHP im Groben.

HTML bildet dabei das Gerüst und CSS ist für die visuelle Darstellung zuständing. PHP ist eine serverseitige Skriptsprache und JavaScript ist für die Interaktion zuständig.
